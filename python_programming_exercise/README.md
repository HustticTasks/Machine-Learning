# Python programming exercise

## Problems
1. 求100万以内的所有素数个数
2. 移除一个列表中的重复元素，并保持列表原来的顺序
3. 扁平化字典
  例如：
  {‘a’: {‘b’: 1}} 扁平化之后是 {‘a.b’: 1}
  {‘a’: {‘b’: {‘c’: 1, ‘d’: 2}, ‘x’: 2}}扁平化之后是{‘a.x’: 2, ‘a.b.c’: 1, ‘a.b.d’: 2}
